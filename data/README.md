# Data

이 디렉터리는 포트폴리오 공개에 적합한 최종 산출물 중심으로 정리했습니다.

## Included

`processed/`에는 분석 결과를 설명하는 데 필요한 집계·점수·등급 파일만 남겼습니다.

- `final_all_data.xlsx`: 주요 유튜버 지표를 결합한 최종 통합 데이터
- `최종_유튜버_데이터.xlsx`: 분석 대상 유튜버의 주요 프로필·성과 데이터
- `월간_구독자_데이터.xlsx`: 월별 구독자 관련 지표
- `감성점수.csv`: 유튜버·월 단위 댓글 감성점수
- `충성도.csv`: 유튜버·월 단위 충성 시청자 비율
- `인지도부문.csv`: 인지도 산정에 사용한 집계 지표
- `영상_간격.xlsx`, `간격_master_df.xlsx`: 업로드 간격 지표
- `등급_예측.xlsx`: 등급 예측 모델링 산출물
- `raw_data_score.csv`: 등급 산정용 점수 테이블

## Excluded

다음 자료는 공개 레포 후보에서 제외했습니다.

- 원천 댓글 데이터
- 유튜버별 원천 영상 CSV 묶음
- 학습된 모델 가중치와 토크나이저
- ChromeDriver 실행 파일
- 대형 중간 산출물

현재 로컬 작업에서는 제외 파일을 상위 폴더의 `_portfolio_excluded_local/`에 보관했습니다. GitHub 공개 시에는 이 폴더를 포함하지 않습니다.

## Note

일부 CSV는 원본 노트북 실행 환경의 인코딩 문제로 컬럼명이 깨져 있습니다. 포트폴리오 문서에서는 분석 목적과 사람이 해석 가능한 핵심 지표명을 기준으로 설명합니다.
